# AI Resume Analyzer 🚀

AI Resume Analyzer is a web-based application that analyzes resumes, calculates job readiness score, identifies skill gaps, and provides personalized recommendations based on selected job roles.

---

## 📌 Project Modules

1. Resume Parsing & NLP
2. Job Role & Skill Dataset
3. Skill Gap Algorithm
4. Readiness Score Logic
5. Recommendation Engine
6. Frontend UI/UX

---

## 🛠️ Tech Stack

Frontend:
- React.js
- CSS
- Axios

Backend (To Be Integrated):
- Node.js
- Express.js
- MongoDB
- NLP Libraries
